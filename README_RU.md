#  Find The Key — игровой прототип на Unity

**Find The Key** — небольшой прототип от первого лица, созданный всего за **2 дня**, демонстрирующий работу основных игровых систем, событийно-ориентированную архитектуру, взаимодействие с объектами и UI-логику.  
Проект создан как портфолио-демо для отклика на вакансии в игровой индустрии.

---

##  Возможности проекта

###  Система взаимодействия
- Взаимодействие игрока через raycast  
- Открытие и закрытие мебели  
- Подсветка объектов через дополнительный материал  
- Проверка дистанции до объекта 

###  Система посадки семян
- Посадка семян в горшки  
- Проверка направления и расстояния  
- Создание новых объектов  
- Обновление прогресса награды  

###  Квестовая система
- Данные квестов на ScriptableObject  
- Пошаговое выполнение  
- События:
  - `OnQuestStarted`
  - `OnFirstQuestFinished`
  - `OnRewardProgress`
- Автоматический переход к следующему квесту  

###  Система награды
- Появление ключевого предмета после выполнения всех шагов  
- UI обновляет прогресс в реальном времени  

###  Аудиосистема
- Собственный компонент SoundManager  
- Фоновая музыка (OGG)  
- Регулировка громкости  

###  Игрок и камера
- Плавный FPS-контроллер  
- Горизонтальный и вертикальный поворот камеры  
- Взаимодействие через центральный луч камеры  

---

##  Основные системы проекта

### **1. PlayerInputSystem**
Отвечает за все взаимодействия игрока с окружающим миром.  
Использует raycast и событийную архитектуру, что исключает прямые зависимости между объектами.

**Основные задачи:**  
- Обнаружение объектов через raycast  
- Подсветка интерактивных объектов  
- События (`whenIOpen`, `onItemPickUp`, `onSow`)  
- Проверка дистанции  
- Связь с **WorldFurniture**, **WorldPot**, **QuestManager**  

Эта система — **связующее звено** между игроком и всеми интерактивными объектами.

---

### **2. WorldFurniture / Interactables**
Модульная система поведения мебели (шкаф, сундук, ящики).  
Каждый объект следует общей схеме взаимодействия, но имеет свои анимации и логику.

![Gif_1_Outline](https://github.com/user-attachments/assets/04ba4945-c53b-420f-a10c-48d8dcd83e2e)

**Основные задачи:**  
- Проигрывание анимаций открытия/закрытия  
- Разрешение/запрет взаимодействия  
- Отправка событий обратно в PlayerInputSystem  
- Поддержка материалов подсветки  

Легко расширяется: новые типы мебели можно добавлять без изменения PlayerInputSystem.

---

### **3. SowManager (система посадки)**  
Отвечает за создание точки посадки и передачу управления системе роста (SeedSow).

![Gif_3_Grow](https://github.com/user-attachments/assets/4b617463-e4ce-4b67-b5e5-e813a1bf3e59)

**Основные задачи:**  
- Подписывается на событие `onSow` из PlayerInputSystem  
- Создаёт объект **PlantEmpty** в месте посадки  
- Назначает ему компонент **SeedSow**, который уже полностью управляет ростом  
- Сообщает QuestManager о прогрессе  
- Cледит только за фактом "посадка выполнена", не участвуя в дальнейшем росте  

SowManager выполняет минимальную связующую логику, оставляя всё поведение роста объекту **SeedSow**, что поддерживает чистую модульность.

---

### **4. QuestManager**
Центральный контроллер линейной квестовой цепочки.

![Gif_2_Progress](https://github.com/user-attachments/assets/d273efff-fc1a-4a39-9140-f58feea27423)

**Основные задачи:**  
- Загружает данные квестов через ScriptableObjects  
- Отправляет обновления UI (`OnQuestStarted`, `OnRewardProgress`)  
- Создаёт ключевые предметы / горшки  
- Управляет машиной состояний квеста  
- Координирует работу SowManager и PlayerInputSystem  

Эта система объединяет все игровые механики в единый цикл.

##  Как запустить проект

1. Клонируйте репозиторий:
git clone https://github.com/PixelNoOne/Find-The-Key-Game.git
2. Откройте проект в **Unity 6.3 LTS**  
3. Откройте сцену **SampleScene**  
4. Нажмите **Play**
