# Find-The-Key-Game
A Unity prototype demonstrating core gameplay systems: character movement, interactable furniture, a simple quest flow, and a reward system built with clean, event-driven architecture.

## Features
- Player movement system  
- Interaction system (doors, furniture, objects)  
- Quest flow: **find → deliver → reward**  
- Inventory interaction  
- Event-driven architecture (clean, modular)

## Tech Stack
- **Unity**  
- **C#** — event-driven systems  
- **ScriptableObjects** for data  
- **Git** for version control  

## How to Run
1. Clone the repository  
2. Open the project in Unity Hub  
3. Press **Play**
